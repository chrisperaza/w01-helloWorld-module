# Hello World Program

## Overview

"Hello World" program written in JavaScript and integrated with Visual Studio Code.

The program's purpose is to demonstrate integration between Visual Studio Code and GitHub, using Git as the version control tool.

["Hello World" Video Demo on YouTube](https://youtu.be/uVscGXpXKIw)

## Development Environment

Node

JavaScript

Git

Visual Studio Code
